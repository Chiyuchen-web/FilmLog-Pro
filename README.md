FilmLog Pro

个人胶片拍摄管理系统，涵盖拍摄到冲洗全流程。

一个面向胶片摄影爱好者的 Web 应用，用于记录拍摄参数、管理冲洗工艺、计时冲洗流程，并支持云端同步与 AI 辅助建议。

# 功能
- 拍摄记录 — 记录胶片型号、画幅（135/120/4x5/8x10）、光圈、快门、闪光灯、拍摄地点及备注，支持图片上传与压缩
- 冲洗记录 — 记录冲洗工艺（C-41/E-6/B&W）、显影液/定影液型号与稀释比、是否预湿、冲洗/定影时间
- 冲洗计时器 — 可配置多阶段计时（预湿→显影→停显→定影→水洗→润湿剂），每个阶段支持独立摇晃/静置参数、倒液时间、循环次数
- 工艺库 — 保存和复用冲洗配方（DevRecipe），支持评分与备注
- 互易律失效计算器 — 支持幂律模型和多项式模型，可按胶片型号保存配置，支持 ND 滤镜换算
- 暗房安全灯模式 — 全界面红色滤镜，适配暗房环境使用
- AI 辅助 — 接入 Google Gemini，根据胶片型号和拍摄环境生成拍摄建议、根据冲洗参数生成冲洗建议
- 云端同步 — 通过 Supabase 实现离线优先的本地存储 + 云端同步，内置同步队列与冲突处理
- 高级搜索 — 多条件筛选（包含/不包含/等于/不等于），支持按日期、型号排序
- 批量操作 — 多选删除、软删除回收站
- 多语言 — 中文 / English 双语
- 深色/浅色主题 — 支持切换，暗房模式独立于主题

# 技术栈

框架 ：React 18 + TypeScript          
构建 ：Vite 6                       
样式 ：Tailwind CSS (CDN)             
图标 ：Lucide React                   
云端 ：Supabase (PostgreSQL + JS SDK)

# 适用场景
胶片摄影师在拍摄和暗房冲洗时，用它来：
1. 记录每一卷的拍摄参数和现场备注
2. 管理和复用冲洗配方
3. 在暗房中使用计时器引导冲洗流程
4. 计算长曝光下的互易律失效修正值
5. 在暗房安全灯下通过红色界面模式查看设备
